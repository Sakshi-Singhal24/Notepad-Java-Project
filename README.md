# Java Notepad Application

A desktop-based Notepad application built using Java Swing that replicates and enhances basic text editor functionalities.
This project focuses on GUI design, event-driven programming, and file handling, providing a hands-on experience of building a real-world desktop application from scratch.


# Features
- Create, open and save files
- Undo and Redo functionality using UndoManager
- Cut, Copy, and Paste operations
- Find, Find Next, and Replace
- Word Wrap toggle
- Font customization (font family, size, bold, italic)
- Live preview for formatting
- Word, character, and line count
- Keyboard shortcuts for efficient usage
- Confirmation dialogs to prevent data loss


# Tech Stack
- Java
- Java Swing (GUI)
- AWT (Event Handling)
- File I/O (FileInputStream, FileOutputStream)


# How to Run
1. Clone the repository  
2. Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)  
3. Run the file: `UseNotepadProject.java`
